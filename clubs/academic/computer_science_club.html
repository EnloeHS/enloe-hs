Computer Science Club

Description

Our club is a place where you can develop websites, games, and more. Tools and information about ways to make sounds, graphics, and code makes Compsci Club a place to learn about creating the frontend, backend, and assets of a project from from start to finish. Come if you want to make any graphics, sounds, or code for applications on our website!

Meetings: 3 Mondays a month, usually 30mins-1hour long
